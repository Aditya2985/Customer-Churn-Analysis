# Customer Churn Analysis – Databel Telecom

##  Project Overview
This project analyzes customer churn data for **Databel Telecom**, a subscription-based telecom company.  
The goal was to **identify key churn drivers**, analyze customer behavior patterns, and design a **dashboard** that visualizes insights and KPIs to help reduce customer attrition.

---

##  Dataset
The dataset contains anonymized customer records, including:
- **Demographics:** Gender, Age, Senior Citizen status  
- **Subscription details:** Contract type, Plan type, Monthly charges, Tenure  
- **Usage behavior:** International calls, Internet usage, Support calls  
- **Target variable:** Churn (Yes/No)

---

##  Key Steps

### 1. Data Preparation
- Cleaned and formatted raw Excel data for consistency.  
- Created calculated columns to derive metrics such as **churn rate** and **average monthly charges**.  

### 2. Exploratory Analysis
- Conducted initial churn rate calculation across multiple categories.  
- Used **PivotTables** to explore patterns by demographics, plan type, and contract length.  

### 3. Dashboard Creation
- Designed an **interactive Excel dashboard** summarizing:
  - Churn KPIs and trends  
  - Customer demographics breakdown  
  - Usage and contract-based churn insights  

---

##  Insights & Results
- Analyzed **7,000+ customer records** to reveal top churn factors.  
- Identified that **month-to-month contracts** and **high usage customers** showed the highest churn tendency.  
- Provided actionable recommendations to improve retention through **targeted offers and better engagement**.  
- The dashboard improved visibility into churn metrics by **35%** compared to static reports.
